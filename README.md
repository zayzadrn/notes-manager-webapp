## Description

This is a Flask-based Notes Manager web application that allows users to create, edit, delete, search, and organize notes using tags.

The application stores notes inside a CSV file and displays them dynamically using Flask and Jinja templates.

Users can:
- Add notes
- Edit notes
- Delete notes
- Search notes
- Filter by tags
- View statistics
- Organize notes into categories

This project was built using Python, Flask, HTML, and CSS.

## Technologies Used
- Python
- Flask
- HTML
- CSS
